# Travel Guide API Documentation

### API Endpoints

- **Register API:**  
  `POST` http://localhost:7000/api/v1/auth/register

- **Login API:**  
  `POST` http://localhost:7000/api/v1/auth/login

- **Logout API:**  
  `POST` http://localhost:7000/api/v1/auth/logout

---

### Registration Form Fields

- **Username**
- **Full Name**
- **Email**
- **Password**

---

### Login Form Fields

- **Email**
- **Password**
