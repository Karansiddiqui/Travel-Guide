# Travel Guide API Documentation

### API Endpoints
### Prefix for api : http://localhost:7000/
- **Register API:**  
  `POST` /api/v1/auth/register

- **Login API:**  
  `POST` /api/v1/auth/login

- **Logout API:**  
  `POST` /api/v1/auth/logout

---

### Registration Form Fields

- **Username**
- **Full Name**
- **Email**
- **Password**

---

### Login Form Fields

- **Email**
